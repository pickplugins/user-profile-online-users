== Description ==

Display online users on your website made easy by this plugin, use via shortcode to display anywhere.

# Shortcodes

<strong>Online users list</strong>
List of online user with thumbnails and total online user counter.

`[online_users]`

<strong>User Online Status</strong>
Check user online/offline status

`[user_online_status user_id="123"]`


# Translation

Please find the 'en.po' for default translation file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/




== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>User profile - Online Users</strong>" activate it.<br />
